Variables

Variables store data values.

They can be declared using let, var, or const.

let age = 20;
  • let
  • var
  • const
Declaring variables

Variables must be declared before use.

Use modern keywords like let and const.

const name = "Ali";
  • Declaration
  • Initialization
  • Assignment
Variable scope

Scope determines accessibility.

There are block and function scopes.

function test() { let a = 10; }
  • Local scope
  • Block scope
  • Function scope
Global variables

Global variables are accessible everywhere.

They are declared outside functions.

var globalVar = 100;
  • Accessible globally
  • Risk of conflicts
  • Stored in window object
Constants

Constants cannot be reassigned.

They are declared using const.

const PI = 3.14;
  • Immutable reference
  • Block scoped
  • Must initialize
Data types

JavaScript has several data types.

Primitive and reference types exist.

typeof "Hello";
  • String
  • Number
  • Boolean
if else statement

The if statement executes based on condition.

Else handles alternative case.

if (x > 5) { console.log("Greater"); } else { console.log("Smaller"); }
  • Conditional logic
  • Comparison
  • Decision making
while statement

The while loop repeats code.

It runs while condition is true.

let i = 0; while (i < 3) { i++; }
  • Loop
  • Repetition
  • Condition check
Function declarations

Functions group reusable code.

They run when called.

function greet(name) { return "Hello " + name; }
  • Reusable
  • Parameters
  • Return values
